# 电商用户价值分层与营销策略优化

> 基于 RFM 模型优化的用户价值评估体系，识别高潜用户群体，提升营销 ROI

## 项目概述

本项目针对电商场景下的用户价值分层问题，在传统 RFM 模型基础上引入行为数据和用户画像，构建了更精准的用户价值评估框架。通过 A/B 测试对比验证，新模型的营销 ROI 是传统策略的 **8 倍以上**。

## 核心亮点

- 引入 4 个新指标：意向度、转化摩擦系数、活跃连接度、购买力背景
- 识别 3 类传统 RFM 看不到的高价值用户：高潜观望用户、隐形活跃者、高潜流失客
- A/B 测试对比：优化策略 ROI 33.52% vs 传统策略 4.05%

## 数据说明

- 数据规模：1000 条用户记录
- 字段数量：15 个
- 数据来源：电商平台用户行为与交易数据

| 字段类型 | 字段名称 | 说明 |
|---------|---------|------|
| 用户画像 | Age, Gender, Location, Income | 基本人口统计信息 |
| 消费数据 | Purchase_Frequency, Total_Spending | 传统 RFM 指标 |
| 行为数据 | Time_Spent_on_Site_Minutes, Pages_Viewed | 网站访问行为 |
| 其他 | Interests, Newsletter_Subscription | 兴趣与订阅状态 |

## 分析流程

### 1. 数据概览

加载数据，检查缺失值，查看描述性统计。

### 2. 探索性数据分析（EDA）

- **关键发现一**：停留时间与购买频率几乎不相关（相关系数 -0.0282），存在大量"观光团"用户
- **关键发现二**：高收入群体中 53.6% 的人消费低于平均水平，存在"沉睡土豪"

### 3. 指标构建

| 指标 | 公式 | 业务含义 |
|------|------|---------|
| 意向度 | 停留时长得分 × 0.5 + 浏览页数得分 × 0.5 | 衡量用户购买意愿 |
| 转化摩擦系数 | 浏览页数 ÷ (购买次数 + 1) | 衡量用户纠结程度 |
| 活跃连接度 | 综合订阅状态与登录行为 | 衡量真实活跃度 |
| 购买力背景 | 基于收入分层 | 区分消费能力差异 |

### 4. RFM 模型构建

对 R（最近登录）、F（购买频率）、M（消费金额）三个维度进行 5 分制评分，计算综合 RFM 分数。

### 5. 用户分层

将用户分为 9 个细分群体：

- 核心 VIP
- 重要价值客户
- 高潜流失客
- 潜力客户
- 高潜观望用户（新发现）
- 隐形活跃者（新发现）
- 新客户
- 沉睡客户
- 普通用户

### 6. A/B 测试对比

| 策略 | 目标用户 | 营销成本 | 边际收益 | 边际 ROI |
|------|---------|---------|---------|---------|
| 方案 A（传统） | 200 人 | ¥2,000 | ¥2,081 | 4.05% |
| 方案 B（优化） | 159 人 | ¥1,590 | ¥2,123 | **33.52%** |

## 核心结论

- 优化策略的边际 ROI 是传统策略的 **8 倍以上**
- 新模型找到的潜力用户对营销刺激敏感度极高
- 高潜观望用户、隐形活跃者是传统 RFM 模型无法识别的宝藏用户
- 每元营销费用能撬动更多增量交易

## 三类高价值新用户

### 高潜观望用户

- 特点：收入高、意向高，但消费金额低
- 运营策略：精准商品推荐 + 优惠激励
- 价值定位：营销投入回报率最高的群体

### 隐形活跃者

- 特点：意向度极高，但收入低、消费低
- 运营策略：鼓励分享和裂变
- 价值定位：口碑传播的种子用户

### 高潜流失客

- 特点：收入高、历史消费高，但近期活跃度下降
- 运营策略：主动触达，提供个性化挽留方案
- 价值定位：挽回成本远低于获取新客户
